Notty
-----

`notty` is a command line tool to detach a process from a tty.


Requirements
------------

 - A compiler (gcc, clang, etc)
 - OS X (tested on 10.10)


Instructions
------------

```bash
make
make install
```


About
-----

The code is originally from http://silmor.de/notty.php, with just minor fixes to work on OS X.
